Asynchronous JediScript - Beta

List and Stream functions are supported.

Available APIs

parallelCounter
Usage: parallelCounter(n)
Descripion: n: number of numbers to count

Count the numbers from 1 to n asynchronously.
Example: 
                        
parallelCounter(10);
                        
//==> 1
//==> 9
//==> 10
//==> 2
//==> 3
//==> 4
//==> 5
//==> 7
//==> 6
//==> 8
parallelAdder
Usage: parallelAdder(lsts)
Descripion: lsts: a list of lists of numbers

Add up a list of lists of numbers asynchronously.
Example: 
                        
var list1 = list(1, 2);
var list2 = list(3, 4);
var list3 = list(5, 6);
var lsts = list(list1, list2, list3);

parallelAdder(lsts);
                        
//==> Current sum: 3
//==> Current sum: 10
//==> Current sum: 21

or

//==> Current sum: 3
//==> Current sum: 14
//==> Current sum: 21

or

//==> Current sum: 7
//==> Current sum: 18
//==> Current sum: 21

etc.
parallelExecute
Usage: parallelExecute(fun1, arg1, fun2, arg2)
Descripion: fun1: first function
fun2: second function
arg1: argument for fun1
arg2: argument for fun2

Run the two functions asynchronously.
Example: 
                        
function square(x) {
  return x * x;
}

function cube(x) {
  return x * x * x;
}

parallelExecute(square, 3, cube, 4);
                        
//==> 64
//==> 9

or

//==> 9
//==> 64
